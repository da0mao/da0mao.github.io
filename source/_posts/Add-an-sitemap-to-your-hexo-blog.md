title: Add an sitemap to your hexo blog
author: Jun Hu
tags:

date: 2019-06-20 22:16:00
---
Install hexo-generator-sitemap plugin
```bash
$ npm hexo-generator-sitemap --save
```
<!-- more -->

Then you can generate the sitemap when you generate your site again 
```bash
$ npm hexo g
```
