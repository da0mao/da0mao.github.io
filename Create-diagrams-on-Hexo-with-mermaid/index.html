Create Diagrams on Hexo With Mermaid

Posted on 17-Aug-2019 Views:

Install package

$ yarn add hexo-filter-mermaid-diagrams

Edit _config.yml

Include mermaid.js in ejs

\themes\icarus\layout\common\footer.ejs

Note: If your theme uses swig or pug files, please refer to here for instruction to include mermaid.js

Start using it on your blog!
